Class IntColorGlider

com.github.yellowstonegames.smooth.Glider
com.github.yellowstonegames.smooth.IntColorGlider
public class IntColorGlider extends Glider
Allows specifying a smoothly-changing color using an int-based color (often RGBA8888, but possibly produced by DescriptiveColor and in Oklab space) for the start and the end, with a change amount that affects how far a color has been edited from start to end. Typical usage sets the start with setStart(int) once and the end more than once with setEnd(int), changing the end each time the move is complete. The current color is automatically calculated in getColor(), and its value will be different every time Glider.setChange(float) is called with a different amount. You can optionally use an Interpolation to make the rate of change different.
This is a type of Glider, and so is compatible with other Gliders (it can also be merged with them).
